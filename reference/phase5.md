# Phase 5: 소독증명서 PDF 생성

## 목표
소독 완료 후 법정 양식 기반 소독증명서 PDF 자동 생성, 다운로드 및 링크 공유

## 선행 조건
- Phase 4 완료 (방문 완료 처리)

## 작업 목록

### 5-1. 증명서 PDF 템플릿
- [ ] 법정 양식 기반 PDF 레이아웃 설계 (`@react-pdf/renderer`)
- [ ] 포함 항목: 업체명, 업체 로고, 소독일시, 소독 장소, 시설명, 사용 약제, 소독 방법, 대표자명
- [ ] 증명서 번호 자동 채번 (CERT-YYYYMMDD-00001 형식)

### 5-2. 증명서 생성 플로우
- [ ] 방문 상세 페이지에 "증명서 생성" 버튼 추가
- [ ] 버튼 클릭 → PDF 즉시 생성
- [ ] 생성된 PDF를 Supabase Storage에 업로드
- [ ] `certificates` 테이블에 기록 (visit_id, certificate_number, pdf_url)

### 5-3. 증명서 뷰어
- [ ] PDF 미리보기 페이지
- [ ] PDF 다운로드 버튼
- [ ] 공유 링크 생성 (Supabase Storage signed URL)
- [ ] 링크 복사 버튼

### 5-4. 발급 이력
- [ ] 증명서 발급 이력 목록 (회사 전체)
- [ ] 고객별 증명서 발급 이력 (고객 상세 페이지 연동)
- [ ] 증명서 재발급 (동일 visit에 대해 재생성)

### 5-5. 업체 설정 — 로고 업로드
- [ ] 설정 페이지에 로고 업로드 기능
- [ ] Supabase Storage에 로고 저장
- [ ] 증명서 생성 시 로고 자동 삽입

- 증명서 발급 이력 관리
